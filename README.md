# Screenshot Tool (Python)

A simple desktop screenshot application built using Python.

##  Features

* Capture fullscreen screenshots
* Capture center region screenshots
* Auto-save with timestamp
* Preview captured image
* Keyboard shortcuts (F1, F2)
* Open screenshots folder

##  Tech Stack

* Tkinter (GUI)
* mss (Screenshot)
* Pillow (Image processing)

##  How to Run

pip install -r requirements.txt
python solution.py

##  Output

Screenshots are saved in the Screenshots folder.
